# 提案：移除整个右侧边栏“设置/概要” Tab

## 动机

在 Willow 桌面应用中，右侧边栏（`ChatRightSidebar.vue`）目前有两个/三个标签页，分别是“文件”、“应用”以及用于展示工作空间信息或会话信息的“设置/概要”标签页。

由于这些基本信息对用户日常任务的操作价值较低，并且占用了右侧边栏的顶部空间，我们决定将这个 Tab 整体彻底移除。移除后，右侧边栏在所有模式下仅展示“文件”与“应用”两个标签页。

## 目标

1. **彻底移除该 Tab**：无论是工作空间模式还是会话模式，都不在顶部导航栏显示“设置”或“概要”的切换按钮。
2. **默认激活文件标签页**：所有模式下，右侧边栏的默认激活标签页（`activeTab`）应自动设为“文件”（`files`）。
3. **清理相关代码**：清理 `ChatRightSidebar.vue` 中仅为“设置/概要”标签页服务的相关响应式状态、计算属性及模板代码。

## 范围

- `app/work/src/renderer/src/pages/chat/components/ChatRightSidebar.vue`

## 非范围

- 其他侧边栏或主聊天区域逻辑。
