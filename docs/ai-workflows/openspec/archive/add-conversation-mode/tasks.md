# 任务列表：新增独立无限对话模式

- [x] 1.1 定义系统级“对话作用域”建模方案，明确是 `workspace.kind`、等价标识还是兼容字段。
- [x] 1.2 设计并补充系统级对话工作区的创建/获取逻辑，默认根目录为用户主目录下的 `.willow`。
- [x] 1.3 梳理会话查询、创建、删除、默认入口逻辑，确保项目工作空间与对话作用域互不干扰。

## 2. 侧边栏与路由

- [x] 2.1 在左侧栏“工作空间”上方新增“对话”分区，并使用非树状单入口呈现。

- [x] 2.3 保持现有项目工作空间树与历史列表行为不回退。

## 3. 对话作用域会话生命周期

- [x] 3.1 实现“获取或创建当前无限会话”的主进程服务逻辑。
- [x] 3.2 确认对话作用域是否禁用标题生成、历史分页入口和会话级下拉操作。
- [x] 3.3 验证 Sender、技能发现、文件选择在 `~/.willow` 根目录下可正常工作。

## 4. 独立上下文压缩

- [x] 4.2 实现长期事实层、未闭环事项层、滚动摘要层、热窗口层的预算与注入策略。

- [x] 4.4 确保普通项目工作空间会话继续沿用现有压缩策略，不被新逻辑误伤。
