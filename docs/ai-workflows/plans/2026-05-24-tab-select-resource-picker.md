# 资源选择面板 Tab 键确认选择 执行计划
   
## 目标
   
支持在提问框的资源选择面板（如斜杠菜单 `/`）打开时，通过按 `Tab` 键快速选中当前被高亮指示的项。
   
## 提出变更
   
1. **输入组件拦截**：
   - `packages/sender/src/components/Sender.vue` -> 在 `handleEditorKeyDown` 处理函数中添加对 `Tab` 按键的拦截。
   
## 验证计划
   
1. **静态检查与格式化**：
   - 运行 `pnpm lint` 验证。
   - 运行 `pnpm build` 验证依赖包与打包。
2. **手动验证**：
   - 运行 `pnpm dev` 启动桌面应用。
   - 在底部输入框中键入 `/`，触发弹出资源选择面板。
   - 使用键盘 `ArrowDown` 移动高亮条目。
   - 按下 `Tab` 键，确认当前条目是否成功插入到输入框中，且输入框恢复聚焦，选择面板正确关闭。
